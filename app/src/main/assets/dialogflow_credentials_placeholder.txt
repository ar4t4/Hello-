DIALOGFLOW CREDENTIALS SETUP INSTRUCTIONS

1. Go to the Google Cloud Console: https://console.cloud.google.com/
2. Create a new project or select an existing one
3. Enable the Dialogflow API for your project
4. Create a service account with Dialogflow Client role
5. Download the JSON key file for the service account
6. Rename the downloaded file to "dialogflow_credentials.json"
7. Place the file in this directory (app/src/main/assets/)
8. Update the PROJECT_ID in DialogflowService.java with your Dialogflow project ID

Note: Keep your credentials secure and never commit them to version control.
